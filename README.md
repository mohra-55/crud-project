# 📋 Contact Hub CRUD Project

## Overview
**Contact Hub** is a web application built with **HTML, CSS, and JavaScript** that allows users to **create, read, update, and delete (CRUD)** contact entries. It includes **form validation** to ensure correct data input and an intuitive interface for managing contacts.

---

## Features
- **Add Contacts:** Create new contact entries with required details.
- **View Contacts:** See a list of all added contacts.
- **Update Contacts:** Edit existing contact information.
- **Delete Contacts:** Remove contacts easily.
- **Form Validation:** Ensures all fields are properly filled.
- **Responsive Design:** Works on both desktop and mobile devices.


---

## Technologies Used
- **HTML** – Structure of the web application
- **CSS** – Styling and responsive design
- **JavaScript** – Handling CRUD operations, validation, and DOM manipulation
